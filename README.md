# Telegram Reminder Bot

![Tests](https://github.com/zhpchshts/telegram-reminder-bot/actions/workflows/tests.yml/badge.svg)

Telegram-бот для создания напоминаний с кастомным расписанием в личных чатах и группах.

Бот рассчитан на использование через Telegram: пользователь добавляет бота в личный чат или группу, задаёт таймзону чата и создаёт напоминания командами.

## Что умеет бот

Бот позволяет:

* создавать одноразовые напоминания;
* создавать повторяющиеся напоминания каждые N дней;
* создавать повторяющиеся напоминания каждые N недель в выбранный день недели;
* создавать напоминания в N-й день недели месяца, например в первый понедельник месяца;
* задавать дату старта для повторяющихся напоминаний;
* настраивать таймзону отдельно для каждого чата;
* изолировать напоминания по чатам: каждый чат видит только свои напоминания и свою таймзону;
* смотреть список активных напоминаний;
* удалять активные напоминания текущего чата;
* использовать бота в личных чатах и группах.


## Команды бота

### Одноразовое напоминание

```text
/remind ГГГГ-ММ-ДД ЧЧ:ММ Текст
```

Пример:

```text
/remind 2026-06-10 12:12 Проверить статус релиза
```

### Повтор каждые N дней

С ближайшего подходящего времени:

```text
/every_days N ЧЧ:ММ Текст
```

Примеры:

```text
/every_days 1 12:12 Каждый день
/every_days 3 12:12 Каждые три дня
```

С кастомной даты:

```text
/every_days_from N ГГГГ-ММ-ДД ЧЧ:ММ Текст
```

Пример:

```text
/every_days_from 3 2026-06-10 12:12 Каждые три дня с 10 июня
```

### Повтор каждые N недель

С ближайшего подходящего дня недели:

```text
/every_week N ДЕНЬ ЧЧ:ММ Текст
```

Примеры:

```text
/every_week 1 SAT 12:12 Каждую субботу
/every_week 2 SUN 12:12 Каждое второе воскресенье
```

С кастомной даты:

```text
/every_week_from N ДЕНЬ ГГГГ-ММ-ДД ЧЧ:ММ Текст
```

Пример:

```text
/every_week_from 2 SUN 2026-06-14 12:12 Каждое второе воскресенье
```

### Повтор в N-й день недели месяца

С ближайшего подходящего дня:

```text
/monthly_weekday N ДЕНЬ ЧЧ:ММ Текст
```

Пример:

```text
/monthly_weekday 1 MON 12:12 Каждый первый понедельник месяца
```

С кастомной даты:

```text
/monthly_weekday_from N ДЕНЬ ГГГГ-ММ-ДД ЧЧ:ММ Текст
```

Пример:

```text
/monthly_weekday_from 1 MON 2026-07-01 12:12 Первый понедельник месяца с июля
```

### Таймзона чата

Показать текущую таймзону чата:

```text
/timezone
```

Установить таймзону чата:

```text
/timezone Asia/Yekaterinburg
```

Примеры:

```text
/timezone Europe/Moscow
/timezone Asia/Almaty
```

Таймзона указывается в IANA-формате. Узнать и скопировать свою таймзону можно здесь:

```text
https://webbrowsertools.com/timezone/
```

Напоминания создаются в таймзоне, которая была установлена в чате на момент создания напоминания.

### Изоляция чатов

Напоминания привязаны к Telegram-чату, в котором они были созданы.

Если бот используется в нескольких личных чатах или группах, каждый чат работает независимо:

* `/list` показывает только напоминания текущего чата;
* `/delete ID` удаляет напоминание только из текущего чата;
* `/timezone` задаёт таймзону только для текущего чата.

Участники одного группового чата могут создавать, смотреть и удалять напоминания этого чата. Напоминания других чатов им недоступны.



### Список активных напоминаний

```text
/list
```

### Удаление напоминания

```text
/delete ID
```

Пример:

```text
/delete 1
```

## Дни недели

В командах используются английские сокращения:

```text
MON, TUE, WED, THU, FRI, SAT, SUN
```

## Локальный запуск

### 1. Создать виртуальное окружение

```powershell
python -m venv .venv
.venv\Scripts\activate
```

Если PowerShell запрещает запуск скриптов, можно временно разрешить выполнение только для текущего окна:

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.venv\Scripts\activate
```

### 2. Установить зависимости

```powershell
pip install -r requirements.txt
```

### 3. Создать `.env`

В корне проекта нужно создать файл `.env`:

```env
BOT_TOKEN=your_telegram_bot_token
APP_TIMEZONE=Asia/Yekaterinburg
```

`BOT_TOKEN` можно получить у Telegram-бота `@BotFather`.

`APP_TIMEZONE` используется как таймзона по умолчанию для чатов, где пользователь ещё не вызвал `/timezone`.

### 4. Запустить бота

```powershell
python bot.py
```

## Технические детали

* Telegram-интеграция реализована на aiogram.
* Планирование напоминаний выполняется через APScheduler.
* Напоминания и настройки чатов хранятся в локальной SQLite-базе.
* При перезапуске приложения активные напоминания восстанавливаются из базы.
* Таймзона сохраняется в напоминание на момент его создания.
* Проверки проекта запускаются через Ruff и pytest.
* GitHub Actions автоматически запускает проверку форматирования, Ruff и pytest при push.
