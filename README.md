# Незабудка

**Незабудка** — Telegram-бот для напоминаний в личных и групповых чатах.

Бот помогает создавать разовые и повторяющиеся напоминания через удобное встроенное приложение Telegram, без длинных команд и ручного ввода расписаний.

## Что умеет бот

* создавать разовые напоминания;
* создавать повторяющиеся напоминания: каждые N дней, каждые N недель, ежемесячно или ежегодно;
* показывать список активных напоминаний;
* редактировать и удалять напоминания;
* учитывать таймзону чата;
* работать в личных и групповых чатах;
* присылать краткую сводку погоды по выбранным населённым пунктам;
* открывать понятный интерфейс прямо внутри Telegram.

## Как начать пользоваться

Откройте бота в Telegram:

[@ZhpchshtsReminderBot](https://t.me/ZhpchshtsReminderBot)

После `/start` бот предложит открыть приложение для управления напоминаниями.

В приложении можно:

1. ввести текст напоминания;
2. выбрать тип расписания;
3. указать дату и время первого срабатывания;
4. проверить предпросмотр;
5. сохранить напоминание.

Созданные напоминания можно изменить или удалить в этом же приложении.

## Типы напоминаний

Бот поддерживает несколько вариантов расписания.

### Один раз

Для разовых событий: встреча, звонок, задача, дедлайн.

### Каждые N дней

Для повторений с произвольным интервалом, например раз в 2 дня или раз в 10 дней.

### Ежедневно

Для регулярных ежедневных напоминаний.

### Еженедельно

Для напоминаний по выбранным дням недели.

### Ежемесячно

Для событий, которые повторяются раз в месяц: например в первый понедельник, во второе воскресенье месяца или в конкретный день месяца.

### Ежегодно

Для дней рождения, годовщин и других ежегодных событий.

## Погода

Бот может присылать краткую сводку погоды на сегодня.

Можно выбрать до 5 населённых пунктов. В сообщении показываются текущая температура, дневной максимум, краткое описание погоды и вероятность осадков.

Данные о погоде берутся из Open-Meteo.

## Использование в групповых чатах

Бота можно добавить в групповой чат.

Напоминания в группе создаются для всего чата. Это удобно для общих событий: дней рождения, встреч, регулярных созвонов, командных задач и договорённостей.

Важно: напоминания в групповом чате не привязаны к отдельному пользователю. Участники чата могут управлять общими напоминаниями этого чата.

## Таймзоны

Бот использует таймзону чата, чтобы корректно рассчитывать время напоминаний.

Таймзону можно изменить в интерфейсе приложения или командой `/timezone`.

Это полезно, если чат живёт в одном часовом поясе, а сервер или отдельные участники находятся в другом.

## Какие данные хранит бот

Для работы напоминаний бот хранит минимальный набор данных:

* идентификатор чата Telegram;
* текст напоминания;
* параметры расписания;
* таймзону чата или напоминания.

Бот не создаёт отдельные профили пользователей и не хранит `user_id` участников групповых чатов.

В личном чате идентификатор чата технически связан с диалогом конкретного пользователя с ботом. В групповых чатах идентификатор относится к чату, а не к отдельному участнику.

## Команды

Основной способ работы с ботом — через встроенное приложение Telegram.

Также доступны команды:

* `/start` — запустить бота;
* `/help` — открыть справку;
* `/app` — открыть приложение;
* `/examples` — посмотреть примеры текстовых команд;
* `/list` — показать активные напоминания;
* `/delete ID` — удалить напоминание;
* `/timezone` — настроить таймзону чата.

Текстовые команды создания напоминаний остаются запасным способом управления. Примеры можно посмотреть через `/examples`.

## Статус проекта

Проект находится в стабильной пользовательской версии. Основные сценарии создания, просмотра, редактирования и удаления напоминаний реализованы.

Новые функции добавляются только при появлении реальной пользовательской необходимости, чтобы не перегружать интерфейс.
